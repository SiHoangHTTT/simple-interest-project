# Contributing Guidelines

Thank you for considering contributing to this project!

## How to Contribute
1. Fork the repository.
2. Create a new branch (e.g., `git checkout -b feature-name`).
3. Make your changes.
4. Commit your changes using a clear message.
5. Push the branch (`git push origin feature-name`).
6. Create a pull request.

## Code of Conduct
Please follow the CODE_OF_CONDUCT.md when interacting with others.

## Reporting Issues
If you find a bug, open an Issue with details.
